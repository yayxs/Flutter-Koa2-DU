## 介绍

### 功能

由于平常不太乐于把自己的想法发表在`大众化的社交平台`，故因此自我开发一`APP`。

- 能够跨端实现相似效果（Android、Ios）
- 实现`想法`，`生活`的记录
- 大致包括：文字、图片、小视频的发布，以及`个人中心`等模块

### 前期须知

这个项目是基于`孤岛」APP`的重新开发，并更换名称`《DU》`,意为`浩瀚宇宙，谁不孤独`,会继续在原有基础开发，并利用`Koa2`开发属于自己的`api`

- 客户端：Flutter
- 服务端：NodeJS
